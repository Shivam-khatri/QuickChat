# QuickChat

QuickChat is a full-stack chat application built using the MERN stack.  
It allows users to register, log in, and communicate through one-to-one chats.

---

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

---

## Features
- User authentication (signup & login)
- One-to-one messaging
- REST APIs for users and messages
- Secure backend architecture
